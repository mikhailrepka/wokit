Component

We have three ways to use pictures in web:

  • Insert <img> with src or <picture> with source
  • Using style="background-image:url('<path>')"
  • Embedding base64 encode picture to code

I'm using first and seconds ways often, that's why I made component to simplify my work.

Picture with caption

Sometimes you need to insert picture with caption, for this you have container .__wo-pict-box.

Picture with caption 
<div class="__wo-pict-box">
	<img src="…" alt="Picture with caption">
	<span>Picture with caption</span>
</div>

Initially picture and caption is on center, but you can use modificators to align picture .--align-left or .--align-right.

Picture with caption aligned left
Picture with caption aligned right 
<div class="__wo-pict-box --align-left">
	<img src="…" alt="Picture with caption aligned left">
	<span>Picture with caption aligned left</span>
</div>
<div class="__wo-pict-box --align-right">
	<img src="…" alt="Picture with caption aligned right">
	<span>Picture with caption aligned right</span>
</div>

Also you can use color modificators, by default is a color --wo-gray. You can use color modificators .--primary and others. Like this:

Picture with caption default
Picture with caption .--primary
Picture with caption .--success
Picture with caption .--info
Picture with caption .--danger
Picture with caption .--warning 
<div class="__wo-pict-box">
	<img src="…" alt="Picture with caption default">
	<span>Picture with caption default</span>
</div>
<div class="__wo-pict-box --primary">
	<img src="…" alt="Picture with caption .--primary">
	<span>Picture with caption <code>.--primary</code></span>
</div>
<div class="__wo-pict-box --success">
	<img src="…" alt="Picture with caption .--success">
	<span>Picture with caption <code>.--success</code></span>
</div>
<div class="__wo-pict-box --info">
	<img src="…" alt="Picture with caption .--info">
	<span>Picture with caption <code>.--info</code></span>
</div>
<div class="__wo-pict-box --danger">
	<img src="…" alt="Picture with caption .--danger">
	<span>Picture with caption <code>.--danger</code></span>
</div>
<div class="__wo-pict-box --warning">
	<img src="…" alt="Picture with caption .--warning">
	<span>Picture with caption <code>.--warning</code></span>
</div>

Of cource you can use <picture> instead <img> for all of this cases.

Plain .__wo-pict element

<picture class="__wo-pict">
	<source srcset="…" media="(max-width:768px)" />
	<source srcset="…" media="(min-width:769px)" />
	<img src="…" />
</picture>

Picture as background

Sometimes you need section with picture as background, this is the way you can made it:

<div class="__wo-pict-bg" style="background-image:url('…');"></div>

You can use .--fixed to fix background position.

<div class="__wo-pict-bg --fixed" style="background-image:url('…');"></div>

Also you can use .--faded for faded background and place content inside of this block.

Header for picture

Lorem Ipsum is simply dummy text of the printing and typesetting industry.

<div class="__wo-pict-bg --fixed --faded" style="background-image:url('…');">
	<h2>Header for picture</h2>
	<p>Lorem Ipsum is simply dummy text of the printing and typesetting industry.</p>
	<button class="__wo-btn --inverted --outline --border-circle">CTA Button</button>
</div>
