Easily create nicely looking buttons, which come in different styles.

Usage

To apply this component, add the .__wo-btn class to an <a> or <button> element. Now you have created a button. Add the color tag to a <button> element to change appearance of the button.

Example

Markup

<button class="__wo-btn">Default</button>
<button class="__wo-btn --primary">Primary</button>
<button class="__wo-btn --success">Success</button>
<button class="__wo-btn --danger">Danger</button>
<button class="__wo-btn --warning">Warning</button>
<button class="__wo-btn --info">Info</button>

Outline

You can use outline variants for all these buttons, add .--outline to any <a> or <button>, and you get outlined element. For example:

Example

Markup

<button class="__wo-btn --outline">Default</button>
<button class="__wo-btn --outline --primary">Primary</button>
<button class="__wo-btn --outline --success">Success</button>
<button class="__wo-btn --outline --danger">Danger</button>
<button class="__wo-btn --outline --warning">Warning</button>
<button class="__wo-btn --outline --info">Info</button>

Inverted

You can use modificator .--inverted at <a> or <button> for blocks with background. It can be applicated ony for default elements. For example:

Example Code Description
.--inverted This modificator added to .__wo-btn for variety of buttons style
.--inverted .--outline This modificatoris good for blocks with a background.

Markup

<button class="__wo-btn --inverted">Default</button>
<button class="__wo-btn --outline --inverted">Primary</button>

Border

Modificator .--border-square set borders to square regardless variable --wo-border-radius-button, modificator .--border-circle set borders to circle.

Example Code Description
.--border-square Set border to square.
.--border-circle Set border to circle

Markup

<button class="__wo-btn --border-square">Square border</button>
<button class="__wo-btn --border-circle">Circle border</button>

Disabled

You can add modificator.--disabled or attribute disabled to <a> or <button> for visual dim element or for disable action function of element.

Example Code Description
class="--disabled" Disable as modificator added to .__wo-btn for visual disabling, as disabled working only with js
disabled Attribute disabled works as usual.
class="--primary --disabled" You can use this modificator or attribute with any button style.

Markup

<button class="__wo-btn --disabled">Disabled</button>
<button class="__wo-btn" disabled>Disabled</button>
<button class="__wo-btn --primary --disabled">Disabled</button>

Sizes

You can use different sizes one one theme. Works with all of styles.

Example

Markup

<button class="__wo-btn --wo-size-xs">Size XS</button>
<button class="__wo-btn --wo-size-s">Size S</button>
<button class="__wo-btn">Without --size-*</button>  <!-- Default size -->
<button class="__wo-btn --wo-size-m">Size M</button>
<button class="__wo-btn --wo-size-l">Size L</button>
<button class="__wo-btn --wo-size-xl">Size XL</button>
