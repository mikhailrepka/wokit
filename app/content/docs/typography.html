Paragraph

Here is paragraph between <p></p>.

You can add class .__wo-notice or element <blockquote> to convert paragraph to notice or blockquote.

Notice example

Blockquote example

<blockquote>Blockquote example</blockquote>

Holders

I'm using holder-elements for markup, this is breakers after floating divs and white spaces there they needed. I call them .__wo-holder-* where * is a number of margin (in rem dividen into ten). For example:

<div class="__wo-holder-0"></div>

Is a breaker with no margin

<div class="__wo-holder-1"></div>

Is a holder with .1rem height.

<div class="__wo-holder-5"></div>

Is a holder with .5rem height.

<div class="__wo-holder-10"></div>

Is a holder with 1rem height.

<div class="__wo-holder-20"></div>

Is a holder with 2rem height.

You can use them in a row.
