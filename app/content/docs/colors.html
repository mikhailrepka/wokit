Color theme.

Usage

I'm using CSS :root for color themes. This is colors for default theme:


Example Class Code Description
.--wo-white #fff Basic white color
.--wo-bg #f4f4f4 Basic background color
.--wo-bg-dim #eae7e7 Background color for elements on basic background
.--wo-black #333333 Basic black color
.--wo-gray-light #cccccc Color for dimmed elements and borders
.--wo-gray #999999 Color for dimmed elements and borders
.--wo-shadow rgba(40,40,40,0.2) Color for shadows
.--wo-inverted #333333 Basic color for inverted elements
.--wo-inverted-dim #000000 Basic color for dimmed inverted elements
.--wo-inverted-light #555555 Basic color for highlighted inverted elements
.--wo-primary #108BB3 Primary theme color for buttons, links and others
.--wo-primary-dim #0E7A9D Dimmed primary color for click/active effects
.--wo-primary-light #1DA0CB Highlighted primary color color for hover effects
.--wo-success #10B362 Color for success indications
.--wo-success-dim #07A758 Dimmed success color for click/active effects
.--wo-success-light #28C778 Highlighted success color for hover effects
.--wo-danger #EF505B Color for danger notification, like errors, empty fields and others.
.--wo-danger-dim #E33F4A Dimmed danger color
.--wo-danger-light #F56872 Highlighted danger color
.--wo-warning #FFA662 Color warning notifications
.--wo-warning-dim #F2964F Dimmed warning color
.--wo-warning-light #FFAE71 Highlighted warning color
.--wo-info #FFD065 Color for notices
.--wo-info-dim #F2C254 Dimmed notices color
.--wo-info-light #F7D17B Highlighted notices color

Color as modificator

You can color every eleemnt using tags .--color-*, like .--color-primary:

Hello world!

<p><span class="--color-danger">Hello</span> world!</p>

Color as background

Also you can use .--bg-* like .--bg-primary for set background color for element

Hello world!

<p class="--bg-black --color-white"><span class="--color-danger">Hello</span> world!</p>
