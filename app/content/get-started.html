Introduction

First of all you need to download WOKit files or include CDN links to your project.
File wo.css is a main style file, key.css for icons support.

WOKit on CDN

All version of wokit available on jsDelivr.net

Markup

<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/mikhailrepka/wokit@latest/lib/key.min.css" />
<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/mikhailrepka/wokit@latest/lib/wo.min.css" />
<script src="https://cdn.jsdelivr.net/gh/mikhailrepka/wokit@latest/lib/wo.min.js"></script>

Self hosted

First download files, then unzip it and include to your code.

Markup

<link rel="stylesheet" href="/path/to/key.css" />
<link rel="stylesheet" href="/path/to/wo.min.css" />
<script src="/path/to/wo.min.js"></script>

Where /path/to/ is folder that locate downloaded files.
